# Frontend Developer Roadmap

Interactive roadmap for learning frontend development — from CRM developer (Aurelia/Vue) to a competitive frontend engineer.

## Features

- 11 structured learning sections covering CS fundamentals to interview prep
- Interactive checklist with progress tracking (saved to localStorage)
- Tips and hints for each topic
- Curated resources with links
- Mobile-friendly sidebar navigation
- Dark theme

## Sections

1. **CS & Web Fundamentals** — how computers, networks, and browsers work
2. **JavaScript Deep Dive** — event loop, closures, prototypes, async patterns
3. **TypeScript** — from basics to advanced type-level programming
4. **CSS & Layout** — flexbox, grid, responsive design, architecture
5. **React** — core concepts, ecosystem, Next.js
6. **Vue 3** — advanced Composition API, Pinia, Nuxt
7. **Tooling & Project Setup** — bundlers, linters, git, project structure
8. **Testing** — unit, integration, e2e, Vitest, Testing Library
9. **DevOps & Deployment** — hosting, nginx, CI/CD, Docker basics
10. **Architecture & Best Practices** — performance, a11y, design patterns
11. **Interviews & Soft Skills** — resume, algorithms, system design, communication

## Usage

Open `index.html` in a browser or host on GitHub Pages. Progress is saved automatically in localStorage.
